/A SUMMARY
This document explains the how to setup cgb.

/B SYSTEM REQUIREMENTS
cgb has the same system requirements as the UCSC Genome Browser.

/C SOFTWARE REQUIREMENTS
cgb requires:

* Apache web server.
* MySQL database server.
* A local install of the UCSC Genome Browser.
  See http://genome.ucsc.edu/admin/mirror.html for a description on 
  how to install the UCSC Genome Browser. You do NOT need to install 
  a full mirror, just the browser. For alternate instructions see 
  http://genomewiki.ucsc.edu/index.php/Browser_Installation.
